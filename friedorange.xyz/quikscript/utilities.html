Utilities

These tools can help with producing Quikscript texts by computer. I originally started this project as a Lua script, but switched to HTML/Javascript as the result would be significantly more user-friendly: everyone already has a web browser, eliminating the need to install additional software; and it allows for easier development of a visual user interface. The programs below donâ€™t require an internet connection to work; feel free to download them to your computer (right-click and choose â€œSave target as...â€ or the option to that effect).

These programs work best in conjunction with a modern keyboard entry method for Quikscript. Furthermore, since they operate offline, you must have a Quikscript font installed on your machine to view any Unicode PUA text.

QSTC (Quikscript Transcoder)

This tool can convert Quikscript texts between the various encoding schemes that have been used over the years. This is especially important when considering that early Quikscript fonts employed custom, and often unique, encodings which doubled as a solution of sorts to the keyboard entry problem (See the character encodings section of the Typing page for further details). Therefore, if you have any old computer-based Quikscript material you wish to resurrect (or new material to set in an old font), QSTC may be of help.
